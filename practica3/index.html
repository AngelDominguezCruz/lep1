Práctica 3: Attiny Digital

Angel Iván Domínguez Cruz
Ingeniería Mecatrónica
Fecha: Septiembre 11, 2023
Programación Attiny

- Resumen -

En esta práctica, exploramos los principios básicos del microcontrolador Attiny85. Aprendimos a configurar nuestro Arduino 1 como programador y a empezar a hacer circuitos básicos usando el Attiny85.

- Introducción -

El Attiny85 es un microcontrolador de tamaño reducido fabricado por Microchip Technology. Es conocido por su versatilidad y bajo consumo de energía, lo que lo hace ideal para aplicaciones con recursos limitados.

El Attiny85 se utiliza en una amplia variedad de proyectos de electrónica y mecatrónica, ya que puede controlar dispositivos, sensores y actuadores, permitiendo la automatización de tareas y la creación de sistemas interactivos. Es una herramienta esencial para ingenieros y entusiastas que desean desarrollar proyectos electrónicos y mecatrónicos compactos y eficientes.

- Materiales -

En esta práctica utilizamos jumpers, diodos LED, resistencias, un protoboard, botones, nuestra placa de desarrollo Arduino Uno y el microcontrolador Attiny85.

- Desarrollo -

El profesor nos mostró una serie de 14 documentos con diferentes circuitos que usan el microcontrolador Attiny85, estos circuitos van desde como configurar el Attiny hasta como prender de distintas maneraas diodos LEDs.

El enfoque de esta práctica era empezar a familiarizarnos con el Attiny85, sus pines y las diferencias con un Arduino normal. En esta práctica solo hicimos circuitos con LEDs muy sencillos.

Simulación

A través de TinkerCad nuestro profesor nos proporcionó las simulaciones de los circuitos con sus respectivos códigos.

Construcción

Hicimos el armado de los 14 circuitos asignados por el profesor, todos los compenentes y diagramas ya habían sido previamente brindados por el profesor.

- Resultados -

Logré realizar de manera efectiva los circuitos brindados por el profesor, además logré configurar mi Arduino como programador y logré usar el Attiny85 de manera efectiva sin ningun error que podría causar algún corto o quemadura dentro el Attiny85. A continuación, los circuitos construidos en esta práctica:

Ejercicio 0: Arduino UNO como ISP

En este ejercicio, configuramos nuestra placa Arduino Uno como programador y dejamos funcional el Attiny85.

Ejercicio 1: Attiny Blink

En este ejercicio, hicimos que un LED parpadeara rápidamente.

Ejercicio 2: Attiny Digital HIGH

En este ejercicio, hicimos que un LED encendiera y se mantuviera en este estado.

Ejercicio 3: Attiny Digital LOW

En este ejercicio, hicimos que un LED se apagara y se quedara en ese estado.

Ejercicio 4: Attiny Digital Delay

En este ejercicio, hicimos que un LED parpadeara con un código propio.

Ejercicio 5: Attiny 2 LEDs Delay

En este ejercicio, hicimos que dos LEDs parpadearan de maneraa asincrónica.

Ejercicio 6: Attiny 3 LEDs Delay

En este ejercicio, hicimos que tres LEDs parpaderan de manera asincrónica.

Ejercicio 7: Attiny Digital Input

En este ejercicio, controlamos el encendido de un LED usando un botón.

Ejercicio 8: Attiny Digital Input IF-ELSE

En este ejercicio, controlamos el encendido de un LED usando un botón y las funciones if y else.

Ejercicio 9: Attiny Digital Input IF variables

En este ejercicio, controlamos el encendido de un LED usando un botón, las funciones if y else y una variable.

Ejercicio 10: Attiny Digital Input OR

En este ejercicio, controlamos el encendido de un LED usando dos botones, estos encendian el LED en función de la compuerta OR.

Ejercicio 11: Attiny Digital Input AND

En este ejercicio, controlamos el encendido de un LED usando dos botones, estos encendian el LED en función de la compuerta AND.

Ejercicio 12: Attiny Contador + 3 LEDs

En este ejercicio, controlamos el encendido de tres LEDs usando un botón, estos encienden de manera sincrónica y se apagan todos de golpe.

Ejercicio 13: Attiny Contador + - 3 LEDs

En este último ejercicio, controlamos el encendido de tres LEDs usando dos botones, estos encienden de manera sincrónica con un botón y se apagan de manera sincrónica con el otro boton.

- Conclusiones -

En conclusión, gracias al programador, el Attiny85 con el lenguaje de Arduino. lo que hace que sea muy fácil de controlar y entender. Los ejercicios realizados en esta práctica fueron muy básicos, pero necesarios para entender el funcionamiento del Attiny85.
